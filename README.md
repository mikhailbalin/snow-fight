## Link to ZEIT Now

```shell
npm i -g vercel
vercel
```

## Environment Variables

Update `env` only with [Vercel](https://vercel.com/mishabalin/snow-fight/settings). To sync run:

```shell
vercel link
vercel env pull
```

## Project setup

```shell
yarn install
```
